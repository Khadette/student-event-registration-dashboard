EduEvents — Student Registration Dashboard

About the Project

EduEvents is a web-based student event registration dashboard designed to help students manage school events easily. The system allows users to register for events, cancel registrations, search for available events, and track seat availability in real time.

Key Features

• View all student events
• Register for an event
• Cancel event registration
• Search events by title or category
• Add new events
• View live statistics
• Store data using Local Storage
